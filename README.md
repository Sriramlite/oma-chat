# Online Messaging Application (OMA)

A modern, responsive online messaging application built with vanilla HTML/CSS/JS and Node.js Serverless Functions.

## Features
- Secure Authentication
- Real-time Messaging (Simulated)
- Dark Mode UI
- Responsive Design

## How to Run Locally

This project is designed to be deployed on Vercel, but you can run it locally using the Vercel CLI.

1.  Install Vercel CLI: `npm i -g vercel`
2.  Run `vercel dev` in the project root.

## Deployment

Push to GitHub and import into Vercel.
"# oma-chat"
